## General info
During Covid-19 time a lot of areas of our lives moved to the internet. This application is one of the solutions to make it simpler to meet people. It enables us to create online events and invite others to join them. Every event has specified interests like _sport, programming, books_ and the URL where the event will take place, so you can easily find the event you're interested in.

## Features
* List of all events
* List of all interests (categories)
* List of events by interest
* List of all participants in a particular event
-------------------------------------------------
* Add new event
* Edit existing event
* Delete existing event
-------------------------------------------------
* Invite friend to an event
* Notification of an upcoming event
